# NLP Analysis of Non-Invasive Heel Spur Mitigation Strategies

This project applies Natural Language Processing (NLP) techniques to analyze research literature related to non-invasive heel spur treatments.

The objective is to identify dominant research themes and treatment approaches using text mining and machine learning.

## Methods Used

- Text preprocessing
- Word frequency analysis
- N-gram analysis
- TF-IDF feature extraction
- Topic modeling (LDA)
- Keyword co-occurrence networks
- Machine learning classification

## Tools

- Python
- Pandas
- Scikit-learn
- NLTK
- NetworkX
- Matplotlib
- WordCloud
